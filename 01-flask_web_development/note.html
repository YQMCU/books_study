* 书名：Flask Web开发：基于Python的Web应用开发实战 * 原名：Flask Web Development:Development Web Application with Python * 作者：[美]Miguel Grinberg * 译者：安道 * 出版社：人民邮电出版社 * 版本：2015年1月第1版 * 备注：图灵程序设计丛书 # Part01 第一部分 Flask简介 ## Chapter01 第1章 安装 [Flask](http://flask.pocco.org/)是一个小型甚至微型框架。 但是，小并不意味着它比其他框架的功能少。Flask被设计为可扩展框架，它具有一个基于服务的强健核心，其他功能则可通过扩展实现。 Flask有两个依赖：路由、调试和Web服务器网关接口（Web Server GateWay Interface,WSGI）子系统由[Werkzeug](http://werkzeug.pocco.org/)提供；模板系统由[Jinja2](http://jinja.pocco.org)提供。 Werkzeug和Jinja2都是由Flask的核心开发者开发而成。 其他诸如数据库访问、Web表单验证和用户认证等高级功能。这些功能以及其他大多数Web程序中需要的核心服务都以扩展形式实现，再与核心包继承。开发者可以任意挑选符合项目需求的扩展，甚至可以自行开发。 **安装Flask**只需要一台安装了Python的电脑。 页面展示
